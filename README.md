# Vs. Dungeons

A dark fantasy RPG digital GM toolkit built with React, TypeScript, and Tailwind CSS.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    # or
    pnpm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    # or
    pnpm dev
    ```

## Features

-   **Character Management:** Create and manage heroes with stats, skills, and inventory.
-   **Skill Trees:** WoW-style talent calculator for professions.
-   **Combat Tracker:** Initiative tracking, health management, and status effects.
-   **Responsive Design:** Optimized for iPad landscape.

## Tech Stack

-   React
-   TypeScript
-   Tailwind CSS
-   Zustand (State Management)
-   Vite
