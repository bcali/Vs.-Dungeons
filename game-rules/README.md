## ðŸ“ Design Principles

2. **Simple math** â€” d20 + bonus. A 7-year-old should never need to calculate more than single-digit addition.
3. **Tactile play** â€” Lego tiles as seals, coins as gold, minifigs as heroes. Physical > digital.
4. **No permadeath** â€” Heroes get knocked out, never killed. Stakes come from story, not fear.
5. **Reward creativity** â€” Bonus XP for clever ideas, funny moments, helping teammates.

7. **Heroes feel heroic from session 1** â€” Starter abilities, Hero Surge, minion enemies, and buffed HP ensure kids never feel helpless.
