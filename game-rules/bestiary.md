## Monster Stat Reference

**Formula:** Level √∑ 2 + 2 = average stat. Specialists deviate ¬±2‚Äì4 from average.
**HP** = TGH √ó 3 (standard) | TGH √ó 4 or 5 (boss)
**MOV** = Spaces per turn on the 10√ó10 grid
**Range** = Attack range in spaces (Melee 1, Reach 2, Short 3‚Äì4, Medium 5‚Äì6, Long 7‚Äì8)

### Range Key
| Code | Meaning |
|------|---------|
| M1 | Melee, 1 space (adjacent) |
| R2 | Reach, 2 spaces |
| S3 / S4 | Short, 3 or 4 spaces |
| Md5 / Md6 | Medium, 5 or 6 spaces |
| L7 / L8 | Long, 7 or 8 spaces |
| X10 | Extreme, full board |

### AoE Key
| Code | Meaning |
|------|---------|
| ST | Single target |
| Rad X | Radius X spaces |
| Line X | Line X spaces long |
| Cone X | Cone X spaces |
| Aura X | Aura X spaces around self |

### Grid Placement Tips

- **Ranged enemies:** Place in rows 8‚Äì10 with melee blockers in rows 6‚Äì7
- **Ambush encounters:** Enemies appear from sides (columns 1 and 10)
- **Boss fights:** Boss in center of back row, minions spread across rows 6‚Äì8
- **Swarm encounters:** Enemies in tight cluster, reward AoE
- **Chase encounters:** Enemies start at row 5, heroes at row 1, objective at row 10
