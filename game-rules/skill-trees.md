## Overview

Every profession has a **skill tree** â€” a branching web of abilities and bonuses unlocked with skill points. Think Diablo talent trees or World of Warcraft specializations.

### How It Works

| Rule | Detail |
|------|--------|
| **Starter Ability** | Every hero gets their profession's first node **auto-unlocked at Level 1** for free |
| **Skill Points** | Earn 1 per level starting at Level 2 (2 at Level 20). Total: 20 points by Level 20 |
| **Spending** | Points go into skills within your profession's branches |
| **Tiers** | Branches have tiers. Spend enough points in a branch to unlock higher tiers |
| **Ranks** | Some skills have multiple ranks (invest more points for stronger effects) |
| **Respec** | **Free, anytime.** Reset your entire tree between encounters, sessions, or during rest |

### Skill Types

| Type | Icon | How It Works |
|------|------|-------------|
| **Active** | âš”ï¸ | Goes in your Action Bar. Use as an action in combat. Free (cooldown-based). |
| **Passive** | ğŸ›¡ï¸ | Always on. No action needed. Permanent bonuses. |

### The Action Bar (5 Slots)

Heroes have **5 action bar slots** for active abilities.

- Only **active skills** (âš”ï¸) and **spellbook spells** go in slots
- **Passive skills** are always on â€” no slot needed
- Swap abilities freely outside of combat
- Normal melee/ranged attacks are always available (no slot needed)

### Skill Trees + Spellbook

The skill tree **extends** the spellbook system â€” it doesn't replace it.

| System | What It Provides | Cost to Use |
|--------|-----------------|-------------|
| **Skill Tree** | Talent tree skills + passives | **Free** (with cooldowns on strong abilities) |
| **Spellbook** | Profession spells (Battle Cries, Techniques, etc.) | **Spell Slots** |
| **Normal Attack** | Basic melee/ranged | **Free** (always available) |

Tree abilities are your "always available" toolkit. Spellbook spells are limited-use power moves.
